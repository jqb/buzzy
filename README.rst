=====
BUZZY
=====

Buzzy is a (very) simple, microframework to generate static sites.

-------
Install
-------

This bit is easy

.. code::

   $ pip install buzzy

done!

-----------
Quick Start
-----------

Let's create your new static site

.. code::

   $ buzzy create my-thing

   $ cd my-thing

   $ buzzy server

done!
