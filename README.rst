=====
BUZZY
=====

Buzzy is a (very) simple, microframework to generate static sites. 

-------
Install
-------

This bit is easy

.. code::bash

   $ pip install buzzy

done!

-----------
Quick Start
-----------

Let's create your new static site

.. code::bash

   $ buzzy start my-site

   $ cd my-site

   $ buzzy server

done!
